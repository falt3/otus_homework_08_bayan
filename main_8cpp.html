Функция сравнения двух файлов поблочно

Parameters
[in]beginИтератор начала диапазона списка файлов для сравнения
[in]endИтератор конца диапазона списка файлов для сравнения
[in]blockSizeРазмер блока чтения из файла
[in]bytesReadКоличество прочитанных байт в файле
[in]groupНомер группы для файлов с одинаковым содержимым

В функцию передается диапазон вектора в котором файлы одинаковы на текущем этапе сравнения. Для каждого файла находится хэш. Файлы сортируются по найденному хэшу, чтобы одинаковые находились рядом. Файлы с одинаковыми хэшами, если не проверились полностью отправляются рекурсивно опять в эту функцию. А файлам, которые полностью проверились и у которых совпал хэш, присваевается уникальный номер группы.
